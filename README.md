# Usage
<!-- usage -->
```sh-session
$ npm install -g heroku-github-plugin
$ oclif-example COMMAND
running command...
$ oclif-example (-v|--version|version)
heroku-github-plugin/0.0.0 darwin-x64 node-v12.21.0
$ oclif-example --help [COMMAND]
USAGE
  $ oclif-example COMMAND
...
```
